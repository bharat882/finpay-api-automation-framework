# FinPay API Automation Framework

A production-style API automation framework built for modern SaaS and FinTech platforms using:

- Postman
- Newman
- JavaScript
- JSON Schema Validation
- GitHub Actions CI/CD

## Project Goals

This framework demonstrates:

- API automation architecture
- Scalable Postman collection design
- Environment management
- Data-driven testing
- Automated regression execution
- CI/CD pipeline integration
- Professional QA engineering practices

## Planned Modules

- Authentication APIs
- Payments APIs
- Invoice APIs
- User APIs

## Tech Stack

| Tool | Purpose |
|---|---|
| Postman | API Testing |
| Newman | CLI Automation |
| JavaScript | Assertions & Logic |
| GitHub Actions | CI/CD |
| JSON Schema | Response Validation |
